1-3 Discuss the five-kingdom classification system and the three-domain theory.
1-3 五界分类系统与三域学说。
- 1969年，美国生态学家惠特克提出五界系统，将生物界分为原核生物界、原生生物界、真菌界、植物界与动物界。
- 1977年，伍斯等人提出生命的三域学说，认为 LUCA 演化出细菌域、古菌域与真核生物域。
- 伍斯和他的同事提出，地球上的生命应当按照系统发生学进行正确分类，然而，长期以来，因原核生物个体微小，很难有完整的化石标本用于分类，因此早期的原核生物分类依据表型进行。
- 而在后来伍斯等人提出用核糖体小亚基 rRNA 序列研究生物的演化关系，通过相似系数的比较，结果将原核生物分为两类，即真细菌与古细菌（为避免混淆，后改称细菌与古菌），开启了原核生物的系统发生学的大门。
- 三域学说的提出是生物分类学与系统发生学研究史上的里程碑事件，通过 rRNA 相似系数的比较弥补先前缺少原核生物化石标本的不足，更好地在分类学中运用了系统发生学的方法，因此三域学说相对五界学说更为进步。

1-3-E The two-domain theory.
1-3-E 两域学说。
- 美国学者雷克根据各类细胞核糖体的研究结果，提出了不同于三域学说的两域学说。
- 该学说认为，真核生物与古菌不是“系统发生关系更近的两个域”，而是属于“同一个域”。
- 伍斯等人提出三域学说时，人们所知道的古菌有限，而后来人们知道的古菌种类丰富了很多。而两域学说在阿斯加德古菌的发现后，又开始得到更多的重视。

### 15 原核生物多样性

15-1 简要叙述细菌形态结构、类型以及在生态环境中的作用，并谈谈它们与人类的关系。
- 形态：细菌的形态多样，最常见的为球状与杆状，此外还有呈弧状的弧菌与螺旋状的螺旋体。
- 结构：细菌的细胞大小变化很大，最小的不到0.1微米，最大的细菌可长0.5cm，肉眼可见。细菌细胞有细胞质膜，外部是细胞壁，但与植物不同，其主要成分是肽聚糖。某些细菌还具有菌毛、鞭毛结构。大多数细菌的细胞质中没有生物膜包被的细胞器，没有明确的分区，但有散在的核糖体。几乎所有细菌细胞质中都有某种形式的内涵体（包涵体）结构。极少数细菌，如蓝细菌含有内膜系统，蓝细菌的细胞质中有由单层膜围成的扁平囊结构——类囊体，是光能吸收和电子传递的场所。细菌没有膜包被的细胞核，其含有染色体的区域被称为拟核，细菌的染色体是环状的，一般一个细胞一个染色体，上面结合少量的蛋白质。不少细菌还有更小的环状DNA，称之为质粒，为染色体外的遗传物质。
- 类型：一，细菌根据染色方法的结果不同可以分为革兰氏阴性菌和革兰氏阳性菌，它们的细胞膜结构有所不同。二，按照细菌获取营养的方式，可分为以下类型：1，光能自养型，以CO2为唯一或主要碳源，以光能为能源；2，化能自养型，以CO2为主要碳源，以化合物（如H2S，NH3）所含的能量为能源；3，光能异养：碳来源于有机物，以光能作为能源；4.化能异养：大多数细菌只能依靠有机物氧化来获取能源和碳源。三，按照细菌代谢与氧气的关系，可分为专性好氧菌，专性厌氧菌，兼性厌氧菌等类别。四，人们现在越来越多地使用基因组学等方法对于已知细菌进行分析，将核酸和蛋白质水平的特征用于其分类和研究，
- 在生态环境中的作用：细菌是自然界的分解者，其分解代谢对自然界的物质循环十分重要，生物的尸体属于复杂的有机物，可以被细菌被分解为简单的化合物如氨基酸、核苷酸后，为植物所吸收利用。在氮循环中，固氮细菌也起着重要的作用，空气中的大量N2，只有少数细菌和真菌才能将其转化为其他生物可以利用的化合物，固氮细菌有独立生活者，亦有与植物共生者如根瘤菌。根瘤菌能够将大气中的氮经固氮酶还原为NH3/NH4+，供植物合成氨基酸之用。
- 与人类的关系：关系密切，在环境污染检测和治理、工农业、医药上广泛应用，也引发多种疾病。

### 22 植物的结构与生殖

22-1 单子叶植物与双子叶植物的主要区别

|鉴别点 | 单子叶植物 | 双子叶植物 |
| -------------- | ---------- | --------- |
| 胚中子叶数量 | 1 | 2 |
| 根系 | 须根系 | 直根系 |
| 茎内维管束 | 排列分散 | 呈环状排列 |
| 叶脉 | 主叶脉平行或呈现弧形 | 叶脉呈网状 |
| 花 | 基数为3 | 基数为4或5 |

其他：
- 单子叶植物的茎原形成层全部分化为初生木质部与初生韧皮部，茎只有初生生长，无次生生长
- 双子叶植物茎初生木质部和初生韧皮部有原形成层保留部分，其茎可进行次生生长，使茎加粗
- 单子叶植物的花粉具有单个萌发孔
- 双子叶植物的划分具有3个萌发孔

22-2 论述被子植物营养器官的结构和功能
- 根(root)、茎(stem)、叶(leaf)是植物的三种营养器官，这三种器官的维管组织有着相互联系，在植物生长中能够相互影响。
- 根系有以下功能:
  （1）吸收作用，吸收土壤中的水分和无机盐，无论是须根系还是直根系都使植物与土壤有很大的接触面积,两类植物的根尖(root tip)都有大量根毛(root hair),又使其表面积大为增加。根毛是根表皮细胞的突起,有利于吸收作用。
  （2）固着和支持作用，机械组织和维管组织使得高大植物能够屹立；
  （3）根还有输导、合成、贮藏、繁殖的功能，还有一定的分泌功能。
- 茎使植物直立在地上,其上有节(node),是叶片的着生处,节之间的茎称为节间(interode)。茎是植株地上部分的主轴，着生并支撑叶、花、果，有输导、支持的作用，还行使贮藏、繁殖的生理功能，尤其在变态茎中，往往存有大量的贮藏物质；不少植物常常以匍匐茎、地下茎或形成不定根或不定芽进行营养繁殖。在有些叶退化的植物当中，如大戟科光棍树，其茎就是其主要的光合作用器官。
- 叶一般有一平展的叶片(blade),以叶柄(petiole)与茎相连。是光合作用和蒸腾作用的部位。植物通过光合作用合成所需要的葡萄糖，并以此为原料看一部合成淀粉、脂肪、蛋白质、纤维素等；蒸腾作用是根系吸水的动力，对矿质元素在植物体内的运转有利；能够降低叶表面温度。使得叶在强烈的日光下，不致因温度升高而受损。叶还有一定的吸收功能，叶面肥可经叶片表面吸收。

22-3 简述植物组织的类型
- 分生组织：具有持续分裂能力，根据分生组织在植物体内的位置不同,可将其分为顶端分生组织(apical meristem)、侧生分生组织(lateral meristem) 和居间分生组织(intercalary meristem)。
- 那些因分化而失去分裂能力,并具有特定结构功能的细胞组织,即成熟组织(mature tissue),又称为永久组织(permanent tissue)。
- 表皮：是一种保护组织。表皮细胞大多扁平。叶、茎的表皮外面有角质层,其上还可能覆盖以蜡质,可防止过分失水,也可以抵御真菌等寄生物侵袭。叶表皮上有气孔,是气体出入的门户,由两个保卫细胞组成。保卫细胞有调节气孔开关的能力。
- 薄壁组织：由薄壁细胞组成,细胞间隙较大。它是大多数植物中数量最多的一种细胞类型。薄壁细胞有多种功能, 如贮藏物质、光合作用、有氧呼吸。可以脱分化，再分化为其他种类的细胞。
- 厚角组织：初生壁比较厚。它的主要功能是为植物生长中的部分提供支持。
- 厚壁组织：厚壁组织的细胞有次生壁,因含有木质素而变得坚硬。成熟后,大多数厚壁细胞死亡,它们的细胞壁成为支持植株的坚强框架。有两种类型的厚壁细胞，一种为纤维(fiber)，另一种为石细胞(sclereid)
- 厚角组织和厚壁组织都属于机械组织(mechanical tissue)。
- 维管组织：又称输导组织,是一个由多种细胞组成的复合组织,常常以束状排列,分为木质部和韧皮部, 通常称为维管束(vascular bundle)。维管束在植物的根、茎、叶中形成一个连续的网络,为植物运送水分和养分,并提供支撑。木质部(xylem)的主要功能是从根部向上运送水分和可溶性矿物质，组成细胞有导管和管胞，韧皮部(phloem)的主要功能是将糖分等有机养分从叶或贮藏组织运送到植物的其他部分，组成细胞有筛管和筛胞。

22-4 举例比较初生生长和次生生长（NJU 2014）
- 位于根尖或茎尖的顶端分生组织及其衍生细胞产生的细胞使植物体(茎和根)的长度增加。这种延长是初生生长(primary growth)的结果,它使根在土壤中广泛分布,使枝叶繁茂,尽可能多地接受阳光和二氧化碳。
- 植物还有次生生长(secondary growth)。次生生长使初生生长所形成的根和茎继续加粗。次生生长的来由是侧生分生组织，侧生分生组织分两类:维管形成层和木栓形成层。
- 大多数双子叶植物和裸子植物的根，在完成初生生长后，由于维管形成层和木栓形成层的活动使根长粗。初生生长使茎和根加长,而次生生长则使它们加粗加固。
- 举例：在木本植物体内,初生生长与次生生长在不同的位置同时发生。初生生长仅限于最幼嫩的部分,例如根尖和茎尖,这里有顶端分生组织。侧生分生组织则在根和茎较老的部分,距顶端稍远处。

- 高等动物的器官比较复杂，如心、肝、脾、肺、肾皆为不同的器官。其中，胃是一种消化器官，由上皮组织、肌肉组织、结缔组织、神经组织构成。

25-3 何谓稳态？从不同层次阐述动物生物学中的稳态（P400）（NJU2014）
- 稳态，最初是由美国生理学家坎农提出的，用于描述机体**代偿性**调节反应所形成的**内环境相对稳定**的状态。稳态并不意味着固定不变，而是指一种可变的相对稳定的状态。
- 对于个体来说，内环境的稳态是细胞生活所必须的，将内环境理化因素（如温度、pH）限制在一个较为狭窄的幅度之内，这是在机体的调节作用下，通过各器官分工合作，协调统一实现的。
- 当然，稳态的概念也有所发展。如体内许许多多的保持协调，稳定的生理过程，从细胞、组织、器官，到系统。例如心脏通过调节血压、血容量、心输出量来维持血液循环的稳定；内分泌系统通过激素调节人体代谢过程，如胰高血糖素和胰岛素调节血糖水平，调节糖代谢的稳态。
- 高于个体的水平也有稳态，在种群、群落，生态系统上，种群的繁殖和死亡维持种群数量稳定，群落水平上有种间关系以维持群落的稳态，而物质循环有利于维持生态系统的稳态。
- 随着分子生物学的发展，分子水平上也拓展了稳态的概念，如基因表达的稳态调节，酶活性的稳态调节。

33-2 简述动物神经系统的演化
- 单细胞与低等多细胞动物（如海绵）无神经系统。
- 最简单的神经系统是神经网，由很细的神经纤维交织而成，没有中枢和外周之分，在刺胞动物（如水螅等）中广泛存在。
- 后来，在演化的过程中，神经细胞的胞体聚集在一起，形成神经节。扁形动物（如涡虫）的神经细胞，初步开始集中成神经节。特别是身体前端的脑神经节和两条腹神经索。
- 环节动物与节肢动物，形成了链状神经系统，神经细胞集中成神经节，神经纤维聚集成束形成神经。神经节之间有神经纤维相联系，形成了一条贯穿全身的神经链。随着刺激的部位和强度不同，动物既可发生局部反应，也可以发生全身性的反应。
- 脊索动物神经系统高度集中，形成了中空的背神经管。
- 在脊椎动物中，背神经管分化为前端膨大的脑和脑后的脊髓。
- 脊椎动物中的哺乳动物中，人类的神经系统是最复杂的神经系统。
- 比较不同动物的神经系统，能明显看到由简单到复杂的进化过程。从整体看，由神经网的全身分布逐渐有集中的趋势，形成神经节和神经索，再进一步形成脑和脊髓。就神经通路来看，最简单的仅涉及一个神经细胞，复杂的有多个神经元参与，并有感觉、中间、运动神经元之分。

### 37 生物与环境

37-1 何为生态因子？有何实际意义？
- 生态因子是指环境中对生物的生长、发育、生殖、行为和分布有着直接影响的环境要素，如温度、湿度、食物、氧气和其他相关生物等。生态因子是生物生存所不可缺少的环境条件，也称生物的生存条件，是环境因子中对生物起作用的因子。
- 1840年，李比希，最小因子法则。
- 实用价值：生态因子的概念具有实用价值。例如，某种植物在特定条件下生长缓慢，或某一动物种群数量增长缓慢，说明有某种生态因子在限制着它们的种群数量增长。在各种生态因子中，并非所有生态因子都有同等的重要性，只有找出可能起到限制作用的生态因子，通过实验确定生物与因子的定量关系，才能解决种群增长缓慢的问题。例如，研究限制鹿种群增长的因子时，发现冬季雪被覆盖地面与枝叶，鹿群觅食困难，此时食物是限制鹿种群增长的生态因子。据此，可在冬季森林中人工增添饲料，提高鹿所能获取的食物数量，进而提高鹿种群的数量。这在生态保护上具有一定的意义。
